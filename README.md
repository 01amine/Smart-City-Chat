# Smart City Chat

**Smart City Chat** is a communication app designed for modern urban environments. It leverages a hybrid mesh network to ensure you stay connected with your city, your neighborhood, and your peers, even when traditional infrastructure fails.

## Getting Started

### Prerequisites
- Xcode 15+
- iOS 16+ or macOS 13+

### Building
1. Clone the repository.
2. Open `SmartCityChat.xcodeproj` in Xcode.
3. Ensure all local packages (`Tor`, `BitLogger`) are loaded.
4. Build and Run on your simulator or device.
